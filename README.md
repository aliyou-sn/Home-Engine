<h3 align="center">Home Engine v1.0</h3>



A custom relay board with esp32 microcontroller, for remote control of appliances based on MQTT protocol,with manual switch feedback on the system.This device was build in other to control home appliances remotely using MQTT being it a light weight protocol,and notify a user if a switch is ON/FF.



## Features
1. remote control 
2. Inbuit MQTT discovery assistant
3. Constant wifi connection
4. Works with Home assistant
5. Running locally

## Hardware Requirements

* ESP32
* Custom relay board by JLCPCB
* HLK-5M05
* Screw terminal
* 5V coil Relay
* 1N4001 relay
* 1K resistor 
* BC572 Transistor
